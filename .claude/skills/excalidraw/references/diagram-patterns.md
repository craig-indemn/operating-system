## Layout Recipes

### Top-Down Flow
- Box width: 180–200px, height: 70–80px
- Vertical gap between rows: 60px (arrow space)
- Center-align boxes at same x coordinate
- Arrows: straight down, `points: [[0,0], [0, gap]]`

### Left-Right Flow
- Same box sizes
- Horizontal gap: 100px
- Same y coordinate for alignment
- Arrows: straight right, `points: [[0,0], [gap, 0]]`

### Hub-and-Spoke
- Central box at (250, 200)
- Satellite boxes arranged in a circle:
  - Top: (250, 0)
  - Right: (500, 200)
  - Bottom: (250, 400)
  - Left: (0, 200)
- Arrows from center to each satellite

### Grid Layout
- Columns: x = 0, 280, 560 (180px box + 100px gap)
- Rows: y = 0, 150, 300 (70px box + 80px gap)
- Works for comparison tables, feature matrices
